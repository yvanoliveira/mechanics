Após importar as bibliotecas sympy e sympy.physics.mechanics, os referenciais e as variáveis são declaradas para dar sequência os cálculos da cinemática.

  1. Referencial inercial:

    N = ps.ReferenceFrame('N')

  2. Referenciais móveis:

    A = N.orientnew('A','Axis',[theta,N.z])
B = A.orientnew('B','Axis',[phi,A.z])
C = B.orientnew('C','Axis',[gamma,B.z])
